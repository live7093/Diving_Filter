# Filter for diving photos
**Module:** ISP-AD23-FS25 | **Authors:** Florian Vollmer & Ruben Schmid | **Date:** 08.05.2025

This project restores the natural colors of diving photos by compensating for depth dependent light absorption (red/orange spectrum loss).

## Features
* **Color Restoration:** Algorithm to reverse underwater color attenuation.
* **Interactive Dashboard:** Real-time filtering via a **Streamlit** interface.
* **Analysis:** Detailed processing pipeline documented in a Jupyter Notebook
